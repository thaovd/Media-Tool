### AIO Media Tool

Phần mềm tổng hợp các công cụ nhanh, tiện lợi dành cho editer, creative

## Tính năng
* **Cắt Video Nhanh**

* **Download Video từ các nền tảng MXH**

* **Tự động tạo subtitle nâng cao sử dụng Google Gemini**

* **Đổi tên file hàng loạt**
